# technical-analysis-stock-market
This project implements various technical indicators to analyze stock trends, including moving averages, Bollinger Bands, RSI, and MACD
📊 Data Sources
Yahoo Finance API (yfinance) for real-time and historical stock prices.
🛠️ Methodology
Extracted stock prices using yfinance.
Implemented Simple & Exponential Moving Averages.
Calculated Bollinger Bands for market volatility.
Derived RSI (Relative Strength Index) to measure overbought/oversold levels.
Plotted MACD (Moving Average Convergence Divergence) for trend analysis.
📌 Key Insights
RSI above 70 signals an overbought market, while below 30 signals oversold conditions.
MACD crossovers indicate bullish/bearish momentum.
Bollinger Bands helped visualize high-volatility breakout points.
🚀 Technologies Used
Python (pandas, numpy, matplotlib, yfinance)
Jupyter Notebook
Yahoo Finance API
